UNIASSELVI - IMPLEMENTAÇÃO DE UMA APLICAÇÃO WEB: Rede Social

Criamos uma rede social como exemplo de uma aplicação web, para isso usamos JSP, Java e Servlets para fazer as paginas de cadastro de usuario e login de usuario. Para criar o feed da rede social usamos Javascript, EJS e Node e seus modulos.

Para utilizar a aplicação é preciso instalar o MySQL, NodeJS, Eclipse, TomCat e o JDK Adoptium.
Instalações dos programas necessários para Windows:
NodeJS - https://nodejs.org/en/download/prebuilt-installer - Instalação comum.                                                                                                     
JDK Adoptium - https://adoptium.net/temurin/releases/?version=11&os=windows&arch=x64&package=jdk - Selecione todos os features disponiveis.                                                                                    
TomCat - https://tomcat.apache.org/download-11.cgi (Windows Service Installer) - Instalação comum.                                                                                    
Eclipse - https://www.eclipse.org/downloads/download.php?file=/oomph/epp/2024-09/R/eclipse-inst-jre-win64.exe                                                                                    
____ 1 - Selecione a opção "Eclipse IDE for Enterprise Java and Web" sem as aspas duplas.                                                                                    
____ 2 - Instalação comum.                                                                                    
MySQL Community - https://dev.mysql.com/downloads/installer/                                                                                    
____ 1 - Seleciona a opção "Custom" sem as aspas duplas no tipo de instalação.                                                                                    
____ 2 - Clique no + em MYSQL Servers > MySQL Server > MySQL Server 8.0 > MySQL Server (versão mais atualizada disponivel), após isso clique na flecha verde virada para direita para colocar na lista de produtos a serem instalados                       
____ 3 - Clique no + em Aplications > MySQL Workbench > MySQL Workbench 8.0 > MySQL Workbench (versão mais atualizada disponivel), após isso clique na flecha verde virada para direita para colocar na lista de produtos a serem instalados             
____ 4 - Clique em Next, Next, Execute, Next, Next, Next, Next                                                                                    
____ 5 - Para a senha do banco de dados, coloque: 1234567                                                                                    
____ 6 - Clique em Next, Next, Execute, Finish, Next                                                                                    
____ 7 - Coloque o nome de usuario "root" e a senha "1234567" ambos sem as aspas duplas, e clique em check                                                                                    
____ 8 - Clique em Next, Execute, Finish, Next, Finish                                                                                    
____ 9 - Abra o MySQL Workbench, vá em MySQL connections e clique com o botão direito em Local instance MySQL80                                                                                    
____ 10 - Clique em Delete Connection e clique em confirmar                                                                                    
____ 11 - Ao lado de MySQL Connections clique no botao com o (+)                                                                                    
____ 12 - Em seguida escreva em Connection name "localhost" sem as aspas duplas, após isso clique em OK                                                                                    

Pronto, tudo instalado para executar o projeto, basta baixar o projeto aqui no github e executar pelo Eclipse o LoginUsuario.jsp ou CadastroUsuario.jsp
